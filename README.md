# Campus Market

Project 6: Campus Marketplace with Crypto Payments

Campus Market is a campus-focused marketplace platform where students can buy and sell products within their college ecosystem using modern web technologies.

---

## Problem Statement

Students often struggle to buy or sell items safely within campus. Public marketplaces lack trust, verification, and campus-level relevance.

Crypto Campus Link solves this by providing a secure, student-friendly marketplace with modern authentication and scalable backend infrastructure.

---

## Features

- Secure authentication using Supabase Auth
- Campus-based marketplace for buying & selling
- Seller-specific product ownership
- Product image upload using Supabase Storage
- Optional crypto wallet integration (Algorand / Pera Wallet)
- Fast and responsive UI with modern tooling
